#Operasi Logika Dasar

Pada pertemuan ini mahasiswa mempelajari konsep dasar operasi logika, yaitu AND, OR, NOT, dan XOR yang merupakan fondasi utama dalam logika matematika dan komputasi. Mahasiswa diperkenalkan pada tabel kebenaran untuk setiap operator logika, kemudian mempraktikkannya menggunakan JavaScript sederhana agar lebih mudah dipahami penerapannya dalam pemrograman.
